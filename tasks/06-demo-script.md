# Phase 6: Demo Script

## Objective

Create a concise demo talk track that sells Agent Passport as the control plane for enterprise AI agents.

## Tasks

### Task 6.1: Opening Pitch

Explain:

- Holiday travel outage
- Customer and revenue impact
- Why autonomous remediation is attractive
- Why ungoverned agents are risky

Deliverable:

- 30-60 second opening script

### Task 6.2: Walkthrough Script

Narrate:

1. Outage detected
2. Incident created
3. Monitoring Agent verified
4. ServiceNow Agent takes over
5. Specialized agents are invoked
6. Each agent presents its passport
7. Root cause is found
8. Risky firewall rollback requires approval
9. Scoped credential is released
10. Service is restored
11. Audit trail proves what happened

Deliverable:

- Step-by-step demo narration

### Task 6.3: Closing Message

Position Agent Passport as:

> The identity, policy, delegation, and audit control plane for enterprise AI agents.

Deliverable:

- 30-second closing script

## Acceptance Criteria

- Demo can be delivered in 5 minutes
- Agent Passport is the hero of the story
- Technical viewers understand how identity is verified
- Business viewers understand the risk and value
